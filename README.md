# Lumiart CRM Online v2 (HTML + Supabase) ⚙️

Это версия под **команду онлайн** + **резерв оборудования по датам**.

## Что добавлено в v2
- Проекты с периодом: start_date / end_date
- Резервы оборудования по датам (таблица `reservations`)
- Серверная проверка доступности через RPC `create_reservation`
- Конфликты резерва считаются в базе (`reservation_conflicts`)
- Задачи привязаны к проекту (выбор проекта из списка)

## Настройка Supabase
1) Создай проект Supabase (EU)
2) Auth: включи Email/Password
3) SQL Editor: запусти `supabase_schema.sql`

## Подключить ключи
В `app.js` замени:
- SUPABASE_URL
- SUPABASE_ANON_KEY

Где взять: Project Settings -> API

## Деплой
Залей папку на Netlify/Vercel/Cloudflare Pages как статический сайт:
- `index.html`
- `styles.css`
- `app.js`

## Про права
Сейчас политика простая: **любой авторизованный** может читать/писать всё.
Для роли "склад/продажи/финансы" можно добавить `profiles` и политики по ролям.
